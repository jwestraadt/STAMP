## Quick Start

**Notebook:** `notebooks/01_quickstart.ipynb`

End-to-end demonstration of the core STAMP workflow using a real dataset of
500 apparent 2-D grain diameters.

### Load measurements

```python
from pathlib import Path
from stamp.io import load

ecds = load(Path("notebooks/data/apparent_diameters.txt"),
            column="ECD_um", unit="µm", label="Grain ECD")
print(f"Loaded {ecds.values.size} measurements, "
      f"range {ecds.values.min():.1f}–{ecds.values.max():.1f} µm")
```

### Descriptive statistics

```python
from stamp.stats import describe

stats = describe(ecds)
print(f"Geometric mean : {stats.gmean.mean:.2f} µm  "
      f"[{stats.gmean.ci_low:.2f}, {stats.gmean.ci_high:.2f}]")
print(f"Median         : {stats.median.median:.2f} µm")
```

### Stereological correction

```python
from stamp.stereo import saltykov, two_step

sal = saltykov(ecds, n_bins=12)
ts  = two_step(ecds, bin_range=(10, 20))

print(f"Two-step geometric mean (3-D): {ts.geometric_mean:.2f} µm")
print(f"Log-shape σ                  : {ts.shape:.3f}")
```

### Plots

```python
from stamp.plot import distribution, saltykov_plot, twostep_plot, qq_plot

distribution(ecds)
saltykov_plot(sal)
twostep_plot(ts)
qq_plot(ecds, distribution="lognormal")
```

## Stereological Correction Validation

**Notebook:** `notebooks/02_simulation_validation.ipynb`

Validates the Saltykov and two-step corrections against a known ground truth
generated by Monte Carlo simulation of the Wicksell corpuscle problem.

### The Wicksell corpuscle bias

When a polished section cuts through a 3-D microstructure, the apparent 2-D
diameters are systematically smaller than the true 3-D grain sizes.
`simulate_section()` reproduces this by drawing random cross-sections from a
synthetic grain pool with known parameters:

```python
from stamp.simulate import simulate_section

sim = simulate_section(
    mu=45.0,          # true geometric mean (µm)
    sigma=0.35,       # log-shape σ of ln D
    n_intersections=500,
    n_grains=10_000,
    seed=42,
)

print(f"True geometric mean    : {sim.mu} µm")
print(f"Apparent geometric mean: "
      f"{sim.apparent_diameters.values.mean():.1f} µm  (biased low)")
```

### Recovery accuracy

```python
from stamp.stereo import two_step
import numpy as np

ts = two_step(sim.apparent_diameters, bin_range=(10, 20))

err_mu    = abs(ts.geometric_mean - sim.mu) / sim.mu * 100
err_sigma = abs(np.log(ts.shape)  - sim.sigma) / sim.sigma * 100

print(f"Geometric mean — true: {sim.mu:.1f} µm, "
      f"recovered: {ts.geometric_mean:.2f} µm, error: {err_mu:.1f}%")
print(f"Log-shape σ        — true: {sim.sigma:.3f}, "
      f"recovered: {np.log(ts.shape):.3f}, error: {err_sigma:.1f}%")
```

Typical output (n = 500 sections):

| Quantity | True | Recovered | Error |
|---|---|---|---|
| Geometric mean (µm) | 45.0 | 43.8 | 2.6 % |
| Log-shape σ | 0.350 | 0.388 | 10.8 % |

### Comparison plot

```python
from stamp.plot import comparison_plot

fig = comparison_plot(sim, ts)   # works with SaltykovResult too
```

### Sample-size effect

Recovery improves with the number of 2-D measurements.
The notebook sweeps n = 100 → 2 000 and plots geometric mean recovery error vs
sample size, showing that n ≥ 500 sections achieves < 5 % error.
