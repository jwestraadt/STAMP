Installation
============

From PyPI
---------

.. code-block:: bash

   pip install stamp

Or with uv:

.. code-block:: bash

   uv add stamp

From Source
-----------

.. code-block:: bash

   git clone https://github.com/jwestraadt/STAMP.git
   cd STAMP
   uv sync --all-extras

Requirements
------------

- Python 3.9 or later
